## Setup Project

You can run this project with firebase for free.

Setup the project with:

`firebase init`

`firebase functions:config:set passwords.hashpassword=<insert custom password here>`


## Available Scripts

In the project directory, you can run:

### `yarn start`

### `yarn test`

### `yarn build`
